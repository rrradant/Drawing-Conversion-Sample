Program Name : Batch (Batch.exe)
Last Modified: February 21, 2017
Visual Studio 2013  VB .Net 4.5

Description
-----------
This sample demonstrates controlling batch processing within Solid Edge.  The sample uses
a form that allows the user to choose three types of batch processing.
	o Printing a directory of Draft files.
	o Converting a set of files into Solid Edge.  For example converting a set of 
	  DXF files into Solid Edge Draft.
	o Converting Solid Edge files into another format.

Operation
---------
The program is run from Explorer and will start Solid Edge if it is not already running.
